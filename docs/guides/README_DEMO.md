# Animus V1 Project Demo 运行指南

本项目是为技术 Leader 准备的 **Animus (具身智能宠物)** 系统演示版本。

## 🌟 演示重点
1. **情感陪伴 (Emotional Companionship)**: 通过内心独白展示 AI 的情感状态。
2. **技术深度 (Technical Depth)**: 实时展示 LangGraph 工作流追踪、RAG 记忆检索过程。
3. **闭环体验**: 模拟传感器输入（触摸、摇晃）与硬件状态反馈（灯光、电机）。

## 🚀 启动步骤

1. **环境准备**:
   确保已激活虚拟环境并安装依赖：
   ```bash
   source venv/bin/activate
   pip install -r requirements.txt
   ```

2. **启动 Ollama (用于本地 RAG 记忆)**:
   确保 Ollama 服务已启动，且已下载模型：
   ```bash
   ollama pull bge-m3
   ```

3. **设置 API KEY**:
   确保环境变量中已设置 `VOLCENGINE_API_KEY`。

4. **运行演示程序**:
   ```bash
   streamlit run demo/app.py
   ```

## 🎮 演示流程建议

### 场景 A：初次见面 (陌生阶段)
- **操作**: 直接输入 "你好"。
- **观察**: 亲密度较低，回复较为礼貌但有距离感。查看“节点追踪”看到路由过程。

### 场景 B：建立连接 (情感互动)
- **操作**: 点击侧边栏的 "摸摸头"。
- **观察**: 亲密度提升。查看“内心独白”，你会看到 Animus 表达被抚摸后的愉悦感。

### 场景 C：深度交流 (记忆与 RAG)
- **操作**: 输入 "我晚上一般几点睡觉？" 或 "我喜欢吃什么？"。
- **观察**: 查看“技术追踪 -> 记忆检索”，系统会从向量库中提取历史记忆。

### 场景 D：边界测试 (冲突处理)
- **操作**: 点击侧边栏的 "摇晃它" 或输入辱骂性词汇。
- **观察**: 亲密度大幅下降，触发“红色警告”灯光，系统进入防御/冷却模式。
